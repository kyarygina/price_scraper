# Price scraper

В проекте реализованы скраперы для получения информации о ценах на товар в разных интернет-магазинах. Также реализовано получение аналитики для сравнения цен на один товар в разных магазинах, а также отслеживание изменения цены на товар в рамках одного магазина.

Команда для установки зависимостей:

```pip install -r requirements.txt```

Команда для запуска скраппинга:

```python3 -m flows.main_flow```

*результат скраппинга записывается в data/raw/prices.csv*

Команды для запуска аналитики:

```python3 analysis/analyze.py```

*результат аналитики записывается в data/processed/analysis.csv и сохраняется в виде графика в data/processed/plot.png*
