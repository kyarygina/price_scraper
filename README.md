# Price scraper

В проекте реализованы скрапперы для получения информации о ценах на товар в разных интернет-магазинах. На основе данных,
полученных в результате скрапинга, можно получить аналитику для сравнения цен на один товар в разных магазинах, а также
отследить изменения цены на товар в рамках одного магазина.
Также в проекте реализовано API для взаимодействия с полученными данными.

Команда для установки зависимостей:

```pip install -r requirements.txt```

Команда для запуска скраппинга:

```python3 -m flows.main_flow```

*результат скраппинга записывается в data/prices.db*

Команда для запуска аналитики:

```python3 analysis/analyze.py```

*результат аналитики записывается в data/processed/analysis.csv и сохраняется в виде графика в data/processed/plot.png*

Команда для запуска API:

```uvicorn api.main:app --reload```

## Использование API

### Получить данные:

/prices

Параметры:

- shop
- min_price
- max_price
- limit
- offset
- sort_by
- order
